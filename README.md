# DroneField X Paraná

Projeto desenvolvido para o Concurso Agrinho 2026.

## Tema

Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.

## Objetivo

Apresentar como drones inteligentes auxiliam a agricultura moderna desde o mapeamento das plantações até a irrigação de precisão.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Dark mode
- Responsividade total
- Simulação de mapeamento
- Detector de pragas
- Calculadora de irrigação
- Dashboard de resultados
- Quiz interativo
- Formulário dinâmico

## Estrutura

- index.html
- style.css
- script.js

## Organização

Código separado em arquivos independentes conforme regulamento.

## Autor

Projeto autoral para fins educacionais.
